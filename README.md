# IBBC Bash Scripting Evaluation 

Pipeline automatizado para processamento de dados paired-end, usando:
- Fastp — limpeza e filtragem de leituras
- FastQC — controlo de qualidade
- GetOrganelle — montagem organelar
- MultiQC — relatório integrado

Scripts criados como avaliação de Introdução à Bioinformática e Biologia Computacional.
